Każdy pojedynczy dysk {% if is_js %}[x, y, r]{% else %}(x, y, r){% endif %} wchodzi do aktywnego zestawu, gdy pionowa linia zamiatania osiągnie współrzędną x x-r i opuszcza aktywny zestaw, gdy linia zamiatania osiągnie x+r. Gdy dysk wchodzi do aktywnego zbioru, należy szukać jego przecięć tylko w tym aktywnym zbiorze.
